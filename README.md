# GOOGLE SHEET TO EXCEL DOCUMENT EXTRACTOR

This is a scripts that extracts documents from google sheet into an excel document. It's a  basic script and easy to work with.

## Installation

### Windows OS

```python
# create a virtual environment
py -m venv venv

# Activate virtual environment
venv\Scripts\activate

# Install dependecies
pip install -r requirements.txt

# Get credentials from the google developers console
# Run the script

py main.py
```

### Linux OS

```python
# create a virtual environment
python3 -m venv venv

# Activate virtual environment
source venv\bin\activate

# Install dependecies
python3 -m pip install -r requirements.txt

# Get credentials from the google developers console
# Run the script

python3 main.py
```
